Alt Text

What is alt text?

Alt text (alternative text) is a textual substitute for visual web page material. While technology is advancing in image depiction, its algorithms are lacking in distinguishing an image’s meaning within the page’s content. Alt text solves this problem by providing both the function and content of the image, making for more cohesive and comprehensive content for all.

Alt text can be expressed in two different ways. First, it can be found within HTML code, specifically the alt attribute within the <img> element. Second, it can also be found near an image through viewable body text, or even on a separate page through linked images and linked text.

Failing to include alt text renders content inaccessible to some users. For instance, visually impaired people may require a screen reader to digest content. When perusing a site, a screen reader turns visual content into audio content, with the page’s content being recited to the user. If a screen reader comes across an image without alt text, the screen reader will be unable to transcribe the image into words, leaving the user without the content’s full scope. It is crucial for all images to have alt text, even if it is merely “null” alt text (alt= “ ”).

 

Why is alt text important for accessibility?

Alt text has multiple purposes. It helps visually and cognitively impaired users by giving screen readers narratable text in the stead of images, enabling users to obtain the quality and meaning of an image. Alt text aids nondisabled users as well, where if images don’t load properly or are blocked by the user, the page displays alt text in the image’s place. Additionally, alt text provides context about the page’s content for search engines, allowing search engine users to find their desired content more efficiently.

Despite its clear benefits to all users, improper usages or complete lack of alt text continues to be one of the largest problems web accessibility encounters. People still incorrectly use alt text, rendering certain content as inaccessible to a significant user base. If more people took the time to learn the ins and outs of alt text implementation, the digital accessibility seen would be much less discriminatory, and digital content would be much more widely accessible for all.

 

What should and shouldn't Alt Text do?

Alt text should…

  1. Be accurate and equivalent in presenting an image’s content and function.

  2. Be succinct yet substantive in expressing an image’s purpose.

    • Sometimes only a few words are needed, but certain circumstances require more. For instance, take the image of Keeanu Reeves and the displayed body text below. Given that the body text already describes Keeanu Reeves and there is no caption labeling the image, the proper alt text should read alt=“Keanu Reeves” as opposed to, for example, alt= “Keeanu Reeves, a celebrity film actor.”

  3. Be used for decorative images and images that don’t offer necessary context to what’s written.

    • In such cases, null alt text (alt=“ ”) should be used. This signifies to both the screen reader and coders that the image is insignificant towards understanding the content. For instance, let’s say you have the below image and the accompanying body text. Since the image of a sunset doesn’t contribute any significant or additional content to the topic, null alt text (alt= “ ”) should be used instead of an actual description, such as alt= “A sunflower patch basking in the sun.”

Alt text shouldn’t…

  1. Be redundant, or give identical information that the text already offers near the image.

    • While redundant alt text is usually improper, there are certain scenarios, such as images with functionality, where redundancy is required. For instance, take the same image of Keeanu Reeves, but let’s have it double as a clickable link along with it having the caption of “Keeanu Reeves, Actor.” While a caption is present, the link function isn’t visually described in the caption. This means the function should be conveyed within alt text, specifically alt=“Keeanu Reeves, Actor,” as opposed to null alt text (alt=“ ”). While redundant, redundancy is needed here to properly detail the linked image’s function.

  2. Contain phrases such as “image of…” or “graphic of…”.

    • Doing so is redundant, for screen readers already announce to the user that something is a “graphic.” The only exception to this rule is if the type of image is important to the content, such as a painting or photograph.
