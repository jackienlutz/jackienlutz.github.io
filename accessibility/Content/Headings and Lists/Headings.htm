How do you add Headings in Madcap Flare?

Headings are important for organizing the content on your page. They also help people with accessibility devices access the content of the page. There are two ways to add headings to your document.

 

Using the style tools

These tools are useful if you are not familiar with coding in HTML, and if you are familiar with programs such as Microsoft Word.

  1. Select the tag selector that you want to turn into a heading.

  2. Click the Styles menu, then click Style Window.

  3. Click on the heading level that you want to use. This alters the selected text to the chosen heading style.

Using the text editor

If you are familiar with the HTML coding language, then the text editor is an easy way to quickly change the heading level of the text.

  1. Click Text Editor.

  2. Locate the text you want to change.

  3. Replace the text within the tags (< >, </ >) with one of the following options: h1, h2, h3, h4, h5, h6. These are the heading levels, and they allow a screen reader to more easily access the content on the page. This step changes the style of the text into a heading.
