Checking for Color Contrast

What are the standards for color contrast?

Web Content Accessibility Guidelines (WCAG) provides standards for contrast ratios–difference between color text and background–that differ based on the needs of different people. WCAG has three levels of conformance that set the requirements for proper contrast ratios: Level A, AA, and AAA. Most companies and organizations aim for Level AA conformance where the contrast ratio for normal text is 4.5:1.

There are many available online resources that can check color contrast and generate colorblind-friendly palettes. The Web Accessibility Evaluation Tool (WAVE) is a widely used tool that analyzes the contrast ratio and accessibility of all elements on a page. The WebAim Color Checker is a quick and intuitive tool to check for color contrast between two colors using hex codes. Adobe recently introduced their Accessibility Tools that analyzes color contrast based on AA or AAA level conformance. One unique feature of the contrast checker is suggesting color pairings that adhere to WCAG standards.

Evaluating Color Contrast

Web Accessibility Evaluation Tool (WAVE) is a free online tool made by WebAim that analyzes color contrast ratios and tests for accessibility best practices. Using icons, WAVE brings attention to areas on a website that do not follow WCAG standards.

  1. Copy the URL address of the website you would like to test.

  2. Navigate to the WAVE Web Accessibility Checker.

  3. Paste the URL address in the Web page address bar. A WAVE report appears.

  4. Select the Contrast tab. A sample of the color checker appears.

  5. Click on a red Contrast Error icon to view details on the contrast ratio and levels of conformance.

  6. In the Detail tab, scroll down to look for the Contrast Error group.

  7. Click on a red contrast icon to view where the contrast error appears on the web page.
